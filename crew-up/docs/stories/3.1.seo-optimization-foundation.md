# Story 3.1: SEO Optimization Foundation

## Status
Done

## Story
**As a** search engine crawler (and indirectly, a producer),
**I want** profile pages to have comprehensive SEO metadata, schema markup, sitemaps, and optimized images,
**so that** profile pages rank highly in search results for "[role] in [city]" queries, enabling organic discovery of crew members.

## Acceptance Criteria
1. Profile pages include enhanced SEO metadata (title, description, OG tags, Twitter cards, canonical URLs)
2. Profile pages include Schema.org JSON-LD markup (Person/Service schema) for rich search results
3. XML sitemap is automatically generated and includes all profile pages
4. Robots.txt file is configured to allow search engine crawling
5. Profile photos use Next.js Image component with proper optimization (width, height, priority, responsive images)
6. Profile pages meet performance targets (<2s load time, LCP optimization)
7. All SEO enhancements follow Next.js App Router best practices
8. All code follows coding standards and project structure
