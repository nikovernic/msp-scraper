### Challenges and Learnings

1. **Route Conflict Resolution:**
   - Discovered Next.js route conflict between `/api/profiles/[slug]` and `/api/profiles/[id]/contact`
   - Solution: Moved contact route to `/api/profiles/[slug]/contact` for consistency
   - Lesson: Next.js requires consistent dynamic segment names at the same route level

2. **TypeScript Inference with React Hook Form:**
   - Encountered type inference issue with multiple `useFieldArray` hooks
   - Solution: Used `as any` type assertion (pragmatic workaround)
   - Lesson: TypeScript's inference can struggle with complex generic types in some scenarios

3. **Test Environment Mocking:**
   - `NextRequest.formData()` and `File.arrayBuffer()` needed explicit mocking in Vitest
   - Solution: Used `vi.spyOn` to mock these methods
   - Lesson: Test environments may not fully simulate browser/Node.js APIs
