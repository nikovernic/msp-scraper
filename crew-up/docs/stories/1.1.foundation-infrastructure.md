### Data Models
[Source: architecture/data-models.md]

**User Model:**
- `id`: UUID (Supabase-generated)
- `email`: string (unique)
- `role`: enum('admin', 'crew')
- `created_at`: timestamp
- `updated_at`: timestamp
- Managed by Supabase Auth with custom role field extension

**Profile Model:**
- `id`: UUID
- `user_id`: UUID (nullable, foreign key to users)
- `name`: string
- `primary_role`: string
- `primary_location_city`: string
- `primary_location_state`: string (2-letter code)
- `bio`: string (nullable, max 250 chars)
- `photo_url`: string (nullable, Supabase Storage URL)
- `contact_email`: string
- `contact_phone`: string (nullable)
- `portfolio_url`: string (nullable)
- `website`: string (nullable)
- `instagram_url`: string (nullable)
- `vimeo_url`: string (nullable)
- `union_status`: enum('union', 'non-union', 'either') (nullable)
- `years_experience`: integer (nullable)
- `secondary_roles`: string[] (nullable)
- `additional_markets`: jsonb (nullable, Array of {city, state} objects)
- `is_claimed`: boolean (default false)
- `claim_token`: string (nullable, unique)
- `claim_token_expires_at`: timestamp (nullable)
- `slug`: string (unique, URL-friendly identifier)
- `created_at`: timestamp
- `updated_at`: timestamp

**Credit Model:**
- `id`: UUID
- `profile_id`: UUID (foreign key to profiles, CASCADE delete)
- `project_title`: string
- `role`: string
- `project_type`: enum('commercial', 'feature_film', 'documentary', 'music_video', 'tv', 'corporate', 'other')
- `year`: integer
- `production_company`: string (nullable)
- `director`: string (nullable)
- `display_order`: integer (default 0)
- `created_at`: timestamp
- `updated_at`: timestamp

**ContactInquiry Model:**
- `id`: UUID
- `profile_id`: UUID (foreign key to profiles, CASCADE delete)
- `producer_name`: string
- `producer_email`: string
- `producer_phone`: string (nullable)
- `message`: string (max 500 chars)
- `shoot_dates`: string (nullable)
- `created_at`: timestamp

### Database Schema
[Source: architecture/database-schema.md]

**Tables to Create:**
1. `public.users` - Extended Supabase Auth table with role field
2. `public.profiles` - Core crew profile data
3. `public.credits` - Crew member work credits
4. `public.contact_inquiries` - Contact form submissions

**Indexes Required:**
- `idx_profiles_slug` on profiles(slug)
- `idx_profiles_location` on profiles(primary_location_city, primary_location_state)
- `idx_profiles_role` on profiles(primary_role)
- `idx_profiles_claimed` on profiles(is_claimed)
- `idx_profiles_claim_token` on profiles(claim_token) WHERE claim_token IS NOT NULL
- `idx_profiles_search` - GIN full-text search index on name, primary_role, bio, location
- `idx_credits_profile` on credits(profile_id)
- `idx_credits_display_order` on credits(profile_id, display_order)
- `idx_contact_inquiries_profile` on contact_inquiries(profile_id)
- `idx_contact_inquiries_created` on contact_inquiries(created_at)

**RLS Policies:**
- Profiles: Public SELECT, authenticated UPDATE for own profile, admin INSERT
- Credits: Public SELECT, authenticated ALL for own profile's credits
- Contact Inquiries: Public INSERT, admin SELECT

**Triggers:**
- `update_updated_at_column()` function
- Triggers on profiles and credits tables for updated_at
